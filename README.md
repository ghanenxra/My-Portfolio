## ✨ Features

- 🎇 **Particle.js Hero** — animated particle background on the landing section
- 🖱️ **Smooth Scrolling** — powered by Sweet Scroll for silky navigation
- 📱 **Responsive Design** — works across all screen sizes
- 🃏 **Modern Card Layout** — for Projects, Achievements, and Certifications
- 🎨 **Custom Gradient Theming** — cohesive purple/indigo color palette throughout
- ⚡ **Zero frameworks** — pure HTML, CSS, and JavaScript

## 🏃 Running Locally

No build step needed — it's plain HTML!

```bash
# Clone the repo
git clone https://github.com/ghanenxra/My-Portfolio.git

# Open in browser
cd My-Portfolio
start index.html   # Windows
# or just double-click index.html
```
